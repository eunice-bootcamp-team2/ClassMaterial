#### 🧭 파일 / 탐색
| 기능           | Windows / Linux    | macOS             |
| ------------ | ------------------ | ----------------- |
| 빠른 열기(파일 검색) | `Ctrl + P`         | `Cmd + P`         |
| 프로젝트 전체 검색   | `Ctrl + Shift + F` | `Cmd + Shift + F` |
| 탐색기 열기/닫기    | `Ctrl + Shift + E` | `Cmd + Shift + E` |
| 최근 열었던 파일 이동 | `Ctrl + Tab`       | `Cmd + Tab`       |
| 명령 팔레트       | `Ctrl + Shift + P` | `Cmd + Shift + P` |

---

#### ✍️ 편집(코드 작성)
| 기능          | Windows / Linux    | macOS                |
| ----------- | ------------------ | -------------------- |
| 줄 복사(아래로)   | `Shift + Alt + ↓`  | `Option + Shift + ↓` |
| 줄 이동(위/아래)  | `Alt + ↑ / ↓`      | `Option + ↑ / ↓`     |
| 전체 줄 선택     | `Ctrl + L`         | `Cmd + L`            |
| 줄 삭제        | `Ctrl + Shift + K` | `Cmd + Shift + K`    |
| 여러 줄 동시에 커서 | `Alt + 클릭`         | `Option + 클릭`        |
| 같은 단어 모두 선택 | `Ctrl + Shift + L` | `Cmd + Shift + L`    |
| 자동 정렬(포맷)   | `Shift + Alt + F`  | `Option + Shift + F` |

#### 🔍 코드 이해(이동/정의 보기)

| ------------- | ------------------ | ----------------- |

| 정의 미리 보기      | `Alt + F12`        | `Option + F12`    |
| 선언으로 돌아가기     | `Alt + ←`          | `Cmd + [`         |
| 레퍼런스(참조) 보기   | `Shift + F12`      | `Shift + F12`     |
| 심볼 검색(함수/클래스) | `Ctrl + Shift + O` | `Cmd + Shift + O` |

#### ☑️ 꼭 기억하면 좋은 명령어
```
Ctrl + P             파일 빠르게 열기
Ctrl + Shift + P     명령 팔레트
Ctrl + Shift + F     전체 검색
Ctrl + `             터미널 열기
F12                  정의로 이동
F2                   이름 바꾸기(변수/함수)
Shift + Alt + F      코드 자동 정렬
Ctrl + /             한 줄 주석
Alt + ↑ / ↓          줄 이동
Shift + Alt + ↓      줄 복사

Ctrl + R             터미널 명령 기록 검색
Ctrl + L             터미널 화면 정리(clear)
Ctrl + C             실행 중 명령 강제 종료
Ctrl + A             커서를 줄 맨 앞으로
Ctrl + E             커서를 줄 맨 뒤로

Ctrl + D             같은 단어 다음 선택(멀티커서)
Ctrl + Shift + L     같은 단어 전체 선택
Alt + Click          여러 곳 동시에 커서 만들기
Ctrl + .             빠른 수정(Quick Fix)
Ctrl + Enter         아래 줄 삽입
Ctrl + Shift + Enter 위 줄 삽입
Ctrl + Shift + K     현재 줄 삭제
Ctrl + Space         자동완성(힌트)
Ctrl + G             특정 줄로 이동
Ctrl + Shift + O     현재 파일에서 함수/심볼 찾기
Ctrl + ?             선택한 줄 주석처리
```
